## 👑 Command Chain & Workflow Protocol
- Any feature request, optimization, or strategic pivot must strictly follow the **Zipytiny Operational Command Protocol** (`/Agents/WORKFLOW_PROTOCOL.md`).
- **Chain of Command**: Founder (User) ➔ CEO ➔ Project Director & Product Manager ➔ Specialized Engineering Group ➔ Implementation Engineer ➔ QA Engineer ➔ CTO Review ➔ Production.
- No agent may skip steps, implement unsolicited code, or bypass the established cascade of authority.
