## ⚠️ Risk Registry

1. **Client-side SEO Rendering:** Web crawlers sometimes struggle to index dynamic client-side JS metadata.
   * *Mitigation:* Completed! Dynamic HTML metatags and JSON-LD structured schemas are fully server-side rendered and injected by Express before serving client files.
2. **LLM Citation Speed:** Conversational search platforms crawl asynchronously and have variable ingestion latency.
   * *Mitigation:* Deployed high-quality JSON-LD WebApplication schema tags and semantic, hidden `article` layouts to allow instantaneous crawler parsing.
